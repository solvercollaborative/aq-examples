What this is

This page demonstrates the AnswerQuestions “AI Guide” — an embeddable assistant that answers a visitor’s questions using a specific body of knowledge. Here, that knowledge is Maven’s publicly available help content about building and running cohort-based courses.

It is an independent prototype built by Solver Collaborative to evaluate how well an AI Guide can help a new creator learn an unfamiliar platform. It is not affiliated with, endorsed by, or operated by Maven.

How it works

The assistant retrieves passages from a curated knowledge base, then composes a grounded answer with citations rather than free-associating. The knowledge base is built by crawling Maven’s public help center, splitting it into passages, and indexing them for retrieval.

Because answers are grounded in that source material, the guide stays close to what the help center actually says — and it can tell you when it doesn’t know. Source content can go stale; always confirm anything time-sensitive against Maven’s current documentation.

Privacy & scope

This demo answers only from public help content. It does not have access to any private Maven account, course roster, or member data, and it does not ask you to sign in.
