AQ Public Embed Example

This page demonstrates the simplest AQ chatbot integration: a public embed using only a product key. No backend code is required.

Try it: Click the chat icon in the bottom-right corner to ask a question.

How It Works

The chatbot widget is loaded by a single <script> tag at the bottom of this page. It connects to the AQ API using your product key, which scopes responses to your tenant's public knowledge base content.

Sample Content

This page belongs to a fictional business. The chatbot can answer questions about the content that has been ingested into its knowledge base.
